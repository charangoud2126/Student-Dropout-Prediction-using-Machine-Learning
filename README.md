# 🎓 Student Dropout Prediction using Machine Learning

## Overview

This project predicts whether a student will drop out or continue their education using machine learning models. It helps educational institutions take early action to improve student retention.

## Problem Statement

Student dropout is a major issue in education systems. Early prediction helps:

* Improve retention rates

* Provide timely intervention

* Support data-driven decisions

## Dataset: student_dropout_dataset.csv

## Features include:

Academic performance

Attendance records

Demographic details

Educational background

Achieved strong prediction performance

Important factors affecting dropout:

Academic performance has the highest impact

Early intervention significantly reduces dropout rates
